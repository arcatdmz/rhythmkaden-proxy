Rhythmkaden-proxy 0.0.3

Yet another HTTP proxy server specialized for augmenting the experience of otoge.net.

Requirements:
Windows or Mac OS X
Kadecot server: http://kadecot.com/

Usage:
1. Update the declaration of variable deviceList (data/otogePlayYouTube.js:L40)
2. Set up your browser to use the HTTP proxy 127.0.0.1:1000
3. Launch launchProxy.cmd (Windows) or launchProxy.sh (Mac OS X)
